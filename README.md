# 线程池模拟器

这是一个用于计算机专业课程学习的线程池模拟器。它通过多线程模拟任务调度、执行和统计，帮助理解线程池的核心概念：

- 任务队列
- 线程（Worker）获取任务并执行
- 任务状态的变化（等待、运行、完成）
- 执行统计与时间线

## 运行方式

```bash
python src/thread_pool_simulator.py --workers 4 --tasks 12 --min 0.5 --max 2.0 --seed 42 --report
```

### 参数说明

- `--workers`：工作线程数量
- `--tasks`：任务数量
- `--min` / `--max`：任务持续时间范围（秒）
- `--seed`：随机种子，便于复现实验
- `--report`：输出每个 worker 的时间线

## 学习建议

- 调整 `workers` 与 `tasks`，观察线程池繁忙程度。
- 修改 `min/max`，观察任务时间波动对总体运行的影响。
- 阅读输出时间线，分析任务在不同 worker 之间分配的过程。
